PROLOGUE
The present page wants to play as descriptive for a PHP class devoted to translate input native language commands for database into MYSQL commands lines.


PURPOSES
The goal of this class is to provide a comfortable code interface to let programmers implement forms within their programs so that users can input self-defined command lines.

The pictures below shows two very trivial pages on how the class can be implemented to retrieve query results from database, either in Italian or in French.

Programmers can freely implement as they want. For example, one suggested way is evidently prompt-like, so that customers can use the program to get their queries.

After coding several programs for local companies and paying attention to customers' requirements, I realized that giving them the power to customize their own researches could definitely yield better performance to the programs themselves, so that the latter ones open to a greater number of possibilities for final users.


HOW THE CLASS WORKS
Basically the class analyzes the input line command, tokenizes it and turns into a sequence of number-codes for further translation. As this new format is given, the translation into the desired languages is then perfomed.
Technically the two passage above:

Original Language -> Number codes -> MySql command line

are performed via a couple of bi-dimensional arrays including each word entry and the associated number code per each row.

As the reader would see in the example code INDEX.PHP, class implementation shall first set up the input language to be handled by standard two letters coding:
  • it : Italian.
  • fr : French.
  • ... : any other language to be supported.
Further operation include the insertion of the original command line, the call to the translation routine, finally the run() function whose only boolean parameter just orders to display or not the results into a table, whether the command line was used to make a selection or a insertion/deletion/update, for example.

TESTING THIS CLASS
Flush all the PHP code into your server document root. The folders distribution shall show up as depicted below:

  • Open your mysql shell.
  • Load the file DBEXAMPLE.SQL via this parametrized command line (fill % fields according to your OS configuration):
    SOURCE %UNIT%\%SERVERROOT%\DBEXAMPLE.SQL
  • Load the file index.php from your browser to see shot 02 coming up on your screen.
